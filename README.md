## 🚀 Как использовать

1. Discord Desktop / Vesktop / Equicord
2. `Ctrl + Shift + I` → Console
3. `allow pasting` → Enter
4. Вставь код из [`discord-orbs-farm-stealth.js`](./discord-orbs-farm-stealth.js)
5. **СТАРТ** → **✕** чтобы закрыть
