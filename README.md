## 🚀 Как использовать

1. Открой Discord Desktop / Vesktop / Equicord
2. Нажми `Ctrl + Shift + I`
3. Включи "Allow pasting"
4. Вставь содержимое файла `discord-orbs-farm-stealth.js`
5. Выбери нужные квесты и нажми **СТАРТ**

## Настройки

- **Режимы работы**: Быстрый / Баланс / Безопасный
- **Автопринятие квестов**
