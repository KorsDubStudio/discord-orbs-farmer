# Discord Orbs Farmer

**v8.0 — Полная перезапись с нуля** ✨

Чистая, современная и максимально стабильная версия фармера Discord Orbs.

## Что это такое?

Userscript, который автоматически выполняет квесты Discord для фарма **Orbs** (видео-квесты и игровые).

Orbs — это награды Discord за прохождение квестов. Их можно обменять на Nitro, предметы и perks.

## 🚀 Установка (просто и быстро)

1. Установи расширение **Tampermonkey** (или Violentmonkey / Userscripts)
2. Открой [discord.com/app](https://discord.com/app)
3. Нажми на иконку Tampermonkey → «Создать новый скрипт»
4. Полностью удали всё содержимое и вставь код из файла `discord-orbs-farmer.user.js`
5. Сохрани (Ctrl+S)
6. Обнови страницу Discord

Готово! Появится красивый интерфейс DisOrbsFarm.

## ⚠️ Важно

- Используй **только для личного использования** и в образовательных целях
- Discord может банить за подозрительную активность — используй на свой страх и риск
- Рекомендуется уровень «Баланс» или «Безопасный»
- Скрипт работает только в веб-версии Discord (не в десктоп приложении)

## Настройки

В UI есть шестерёнка ⚙ — там можно:
- Выбрать уровень скорости
- Включить/выключить автопринятие
- Включить автоклейм наград (рискованно)
- Отключить паузы (рискованно)
- Настроить скорость видео
